Xget provides a unified acceleration layer for virtually every major package manager and software registry. Whether you work with Node.js, Python, Java, Go, Rust, Ruby, PHP, .NET, R, Perl, or TeX, you can point your package manager at Xget and immediately benefit from edge caching and faster downloads — with no changes to your application code.

## Supported registries

| Registry | Prefix | Original URL |
|---|---|---|
| npm | `npm` | `https://registry.npmjs.org` |
| PyPI | `pypi` | `https://pypi.org` |

| conda (community channels) | `conda/community` | `https://conda.anaconda.org` |

## Configuration by ecosystem

<Tabs>
  <Tab title="npm / Node.js">
    **Temporary use**

    ```bash
    # Use Xget for a single install
    npm install --registry https://xget.xi-xu.me/npm/
    ```

    **Global configuration**

    ```bash
    # Set globally for npm
    npm config set registry https://xget.xi-xu.me/npm/

    # Verify
    npm config get registry
    ```

    **Project-level `.npmrc`**

    ```bash
    # Configure project-level mirror (.npmrc is shared by npm and Bun)
    echo "registry=https://xget.xi-xu.me/npm/" > .npmrc

    # Install dependencies
    npm install
    ```

    **Bun — `bunfig.toml`**

    ```toml
    # bunfig.toml (project-level) or ~/.bunfig.toml (global)
    [install]
    registry = "https://xget.xi-xu.me/npm/"
    ```

    ```bash
    # Install dependencies with Bun
    bun install
    ```

    ```bash
    EOF
    ```

  <Tab title="Jenkins">
    **Via Jenkins web interface**

    1. Log in to the Jenkins management interface.
    2. Go to **Manage Jenkins** → **Plugins** → **Advanced**.
    3. In the **Update Site** section, set the URL to `https://xget.xi-xu.me/jenkins/update-center.json`.
    4. Click **Submit** to save.

    **Via configuration file**

    ```bash
    # Edit $JENKINS_HOME/hudson.model.UpdateCenter.xml
    sudo nano /var/lib/jenkins/hudson.model.UpdateCenter.xml
    # Set: <url>https://xget.xi-xu.me/jenkins/update-center.json</url>

    # Restart Jenkins
    sudo systemctl restart jenkins
    ```

    **Download plugins directly**

    ```bash
    # Download Maven plugin
    wget https://xget.xi-xu.me/jenkins/download/plugins/maven-plugin/3.27/maven-plugin.hpi
    ```
