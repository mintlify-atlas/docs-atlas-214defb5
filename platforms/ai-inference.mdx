Xget accelerates AI inference API calls through the `ip/[provider-prefix]` URL scheme. By setting your SDK's `base_url` (or equivalent) to an Xget endpoint, all API requests — including streaming completions, embeddings, and multimodal calls — are routed through Xget's edge network before reaching the upstream provider. Your API keys are forwarded directly to the upstream provider; Xget does not intercept or store them.

<Tabs>
  <Tab title="Python">
    **OpenAI SDK**

    ```python
    from openai import OpenAI

    client = OpenAI(
        api_key="your-api-key",
        base_url="https://xget.xi-xu.me/ip/openai/v1",  # Use Xget
    )

    response = client.responses.create(
        model="gpt-5.1",
        input="Hello, GPT!",
    )

    print(response.output_text)
    ```

    **Anthropic SDK**

    ```python
    from anthropic import Anthropic

    client = Anthropic(
        api_key="your-api-key",
        base_url="https://xget.xi-xu.me/ip/anthropic",  # Use Xget
    )

    message = client.messages.create(
        model="claude-sonnet-4-5",
        max_tokens=256,
        messages=[
            {
                "role": "user",
                "content": "Hello, Claude!",
            }
        ],
    )

    print(message.content[0].text)
    ```

    **Google GenAI SDK (Gemini)**

    ```python
    from google import genai
    from google.genai import types

    client = genai.Client(
        api_key="your-api-key",
        http_options=types.HttpOptions(base_url="https://xget.xi-xu.me/ip/gemini"),  # Use Xget
    )

    response = client.models.generate_content(
        model="gemini-3-pro-preview",
        contents="Hello, Gemini!",
    )

    print(response.text)
    ```

## Multi-provider unified interface

The OpenAI-compatible interface is supported by many providers. You can use the OpenAI SDK to talk to multiple providers simultaneously by changing only the `base_url` and `api_key` per request.

```python
from openai import OpenAI

providers = [
    ("Cohere",  "your-cohere-api-key",  "/cohere/compatibility/v1", "command-a-03-2025"),
    ("Mistral", "your-mistral-api-key", "/mistralai/v1",            "mistral-medium-latest"),
    ("xAI",     "your-xai-api-key",     "/xai/v1",                  "grok-4"),
]

for name, key, path, model in providers:
    client = OpenAI(api_key=key, base_url="https://xget.xi-xu.me/ip" + path)  # Use Xget
    response = client.chat.completions.create(
        model=model,
        messages=[{"role": "user", "content": f"Hello, who are you?"}],
    )
    print(name, "=>", response.choices[0].message.content)
```

<Note>
  Xget passes streaming responses through transparently. Server-sent event (SSE) streams, chunked transfer encoding, and delta-format completions all work without buffering or modification.
</Note>

<Note>
  Your API keys are passed directly to the upstream provider in the `Authorization` header. Xget acts as a transparent proxy and does not log, store, or inspect authentication credentials.
</Note>
